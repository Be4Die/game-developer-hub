# Каналы привлечения

Каналы привлечения — способы, которыми пользователи узнают о продукте и попадают в него.

---

## Разработчики игр

| Канал | Описание |
|-------|----------|
| **Конференции** | Геймдев-мероприятия, где присутствует команда платформы. Прямой контакт, презентации, нетворкинг. |
| **Telegram-канал** | Официальный канал платформы с новостями, анонсами, кейсами. Разработчики подписываются и следят за обновлениями. |
| **Сарафанное радио** | Рекомендации внутри сообщества РУ геймдев-предпринимателей и разработчиков. Усиливается личным пиаром CEO платформы. |
| **Прямое приглашение** | Команда платформы выходит на перспективных разработчиков напрямую — через личные сообщения, знакомства, партнёрские предложения. |

### Воронка

```
Конференция / TG-канал / Сарафан
         ↓
   Интерес к платформе
         ↓
   Контакт с командой
         ↓
   Регистрация в Game Developer Hub
         ↓
   Публикация первой игры
```

---

## Модераторы платформы

| Канал | Описание |
|-------|----------|
| **Внутреннее внедрение** | Модераторы — сотрудники платформы. Канал привлечения — решение руководства о переходе на новый инструмент. |

### Воронка

```
Решение о внедрении Game Developer Hub
         ↓
   Онбординг модератора
         ↓
   Работа в системе
```

---

## Вывод

Основные каналы привлечения разработчиков — конференции, Telegram-канал платформы и сарафанное радио внутри геймдев-сообщества. Важную роль играет личный бренд CEO и прямые приглашения перспективных разработчиков. Модераторы попадают в систему через внутреннее внедрение как сотрудники платформы.
