# Целевая аудитория

## Категории пользователей

Система предназначена для двух категорий пользователей:

| Категория | Описание |
|-----------|----------|
| **Разработчики игр** | Инди-разработчики и небольшие студии (до 5 человек), ориентированные на веб-игры. Основные пользователи системы — создают, публикуют и сопровождают игры на платформе. |
| **Модераторы платформы** | Сотрудники платформы, проверяющие игры на соответствие требованиям перед публикацией. Обеспечивают качество контента. |

---

## Разработчики игр

### Формальные признаки

| Параметр | Значение |
|----------|----------|
| **Тип** | Инди-разработчики, небольшие студии (1–5 человек) |
| **Локация** | Преимущественно СНГ на старте, но без жёсткой привязки — платформа мультиязычная |
| **Возраст** | 18–35 лет |
| **Пол** | Преимущественно мужчины |

### Платежеспособность

| Аспект | Описание |
|--------|----------|
| **Вход на платформу** | Бесплатный, любой может начать |
| **Серверы** | Тарификация за использование серверов платформы; свои серверы — бесплатно |
| **Монетизация игр** | Комиссия с рекламы и внутриигровых платежей |
| **Особые условия** | Для топовых разработчиков возможно партнёрство, при котором платформа выступает издателем и берёт курирование |

### Уровень опыта

Средний и выше. Платформа не позиционируется как стартовая площадка для новичков. Типичный сценарий:

- Разработчик уже запускал игры на других площадках
- Или приходит на основе эксклюзивного партнёрства (топовые проекты)

### Хобби и интересы

Геймдев для этой аудитории — это работа-хобби или чистое хобби. Характерные интересы:

- Разработка игр
- Игра в игры
- Программирование
- Геймдизайн
- 3D моделирование
- Рисование
- Написание сценариев

Каждый разработчик обычно имеет уклон в один из аспектов геймдева (код, арт, дизайн, нарратив).

## Модераторы платформы

### Формальные признаки

| Параметр | Значение |
|----------|----------|
| **Тип** | Сотрудники платформы (штат) |
| **Количество** | 1 профильный (фултайм), периодически помогает второй |
| **Локация** | СНГ |
| **Возраст** | 18–35 лет |
| **Пол** | Преимущественно мужчины |

### Профиль

По бэкграунду ближе всего к:

- **QA-специалист** — проверка качества, тестирование
- **Сотрудник поддержки** — коммуникация с пользователями, обработка обращений

### Хобби и интересы

Совпадают с разработчиками: интерес к играм, геймдеву, технологиям.

---

## Вывод

Основная целевая аудитория — инди-разработчики и небольшие студии веб-игр с опытом, ищущие удобную платформу для публикации и сопровождения проектов. Аудитория платежеспособна, технически грамотна и ценит скорость итераций. Модераторы — внутренние сотрудники платформы с профилем QA/поддержки, работающие с потоком заявок на публикацию.
