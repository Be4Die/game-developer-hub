# Game Developer Hub

Веб-платформа — единое рабочее пространство для публикации и сопровождения веб-игр.

## Что это

Инструмент для разработчиков и модераторов игровой платформы Welwise Games. Заменяет ручные процессы через GitHub, Telegram и настольные программы одним интерфейсом.

**Ключевые возможности:**
- Публикация игр с автоматическим развёртыванием
- Модерация через встроенную систему тикетов
- Управление серверами онлайн-игр
- Аналитика и финансовая отчётность
